# Football System

---

## Jak uruchomić aplikację

### Wymagania:

- **.NET 8**
- **Node.js 18+**

---

### 1 Uruchom Backend

Otwórz terminal i wykonaj:

cd backend/FootballSystem/FootballSystem.Api
dotnet run

Backend uruchomiony na: `http://localhost:5217`

---

### 2 Uruchom Frontend

Otwórz nowy terminal i wykonaj:

cd frontend
npm install
npm run dev

Frontend uruchomiony na: `http://localhost:5173`

---

### 3 Otwórz w przeglądarce

Wejdź na: **http://localhost:5173**

---

## Gotowe!

Aplikacja powinna działać. Zobaczysz listę zawodników.

## Technologie

**Backend:** .NET 8, ASP.NET Core, Entity Framework Core, SQLite  
**Frontend:** React, TypeScript, MobX, React Router, Tailwind CSS
